# CodeAlpha FAQ Chatbot

An AI-powered FAQ Chatbot developed using Flask and Scikit-learn.

## Features

- NLP-based FAQ matching
- TF-IDF Vectorization
- Cosine Similarity
- Interactive Chat Interface
- Fast Responses

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Scikit-learn

## How to Run

```bash
pip install -r requirements.txt
python app.py
```

Open:

http://127.0.0.1:5001
